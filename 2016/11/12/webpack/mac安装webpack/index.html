mac安装webpack

发表于 2016-11-12 分类于 Webpack 阅读次数：

1、用npm的方式安装，首先你要安装node.js,地址下载安装
2、安装完看有没有安装成功，看下版本,打开终端输入：node -v【查看node版本】、npm -v【查看npm版本】
3(必)、在终端里输入： sudo -s 回车，输入你的开机密码回车
4、全局安装webpack：npm install webpack -g 回车
